# Secure Vault App

Aplikacja desktopowa do bezpiecznego przechowywania plików z szyfrowaniem i profilami użytkowników.

## Wymagania
- Node.js (LTS)
- Electron

## Uruchomienie
1. Zainstaluj zależności: `npm install`
2. Uruchom aplikację: `npm start`
